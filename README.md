# 📡 Collector Radar V6

Multi-platform product search engine built with Streamlit.

## Features

- Yahoo Auction scraping
- eBay scraping
- Shopee (best-effort)
- Mercari (placeholder)
- Sorting system
- Multi-platform filter
- Clean UI cards

## Tech Stack

- Streamlit
- Requests
- BeautifulSoup
- SQLAlchemy
- Pandas

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
